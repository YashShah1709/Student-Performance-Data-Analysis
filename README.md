# Student Performance Data Analysis

This project analyzes a student performance dataset to understand how lifestyle, family environment, and access to resources affect academic outcomes.

## Dataset
Student Performance Dataset (Kaggle)

## Objectives
- Identify factors influencing student grades
- Analyze relationships between lifestyle and academic performance
- Generate insights using statistical analysis and visualization

## Analysis Performed
- Descriptive statistics
- Data visualization (bar charts, scatter plots, box plots)
- Relationship analysis between variables

## Key Insights
- Students with internet access tend to achieve higher grades
- Strong family relationships correlate with better academic performance
- Higher alcohol consumption is associated with lower grades
- Family environment plays an important role in student success

## Tools Used
- Excel / Data Visualization
- Statistical Analysis
